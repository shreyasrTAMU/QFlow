To show tc:          tc qdisc show dev wlo1    (wlan0 for openwrt router)
To delete tc:         tc qdisc del dev wlan0 root
To show packet information in queues:				 tc -s qdisc show dev wlan0	(s: stats)


tc qdisc replace dev wlan0 root pfifo
tc qdisc replace dev wlan0 root handle 1: htb default 20
tc class add dev wlan0 parent 1: classid 1:1 htb rate $1 ceil $1
tc class add dev wlan0 parent 1:1 classid 1:10 htb rate $1 ceil $1
tc class add dev wlan0 parent 1: classid 1:20 htb rate 0.5mbps ceil 0.5mbps
tc qdisc add dev wlan0 parent 1:10 handle 10: netem loss 0% delay 0ms
tc qdisc add dev wlan0 parent 1:20 handle 20: sfq perturb 10

tc filter add dev wlan0 parent 1:0 protocol ip prio 1 u32 match ip dst 192.168.1.129/32 flowid 1:10
tc filter add dev wlan0 parent 1:0 protocol ip prio 1 u32 match ip dst 192.168.1.132/32 flowid 1:10
tc filter add dev wlan0 parent 1:0 protocol ip prio 1 u32 match ip dst 192.168.1.145/32 flowid 1:10
tc filter add dev wlan0 parent 1:0 protocol ip prio 1 u32 match ip dst 192.168.1.123/32 flowid 1:20

3. Ryu:
testryu.py on nuc4
To install:             https://ryu.readthedocs.io/en/latest/getting_started.html

Running ryu:        ryu-manager --verbose fb_exec_cmd.py
Starting openflow on router:  /etc/init.d/openflow start
Stopping openflow             :  /etc/init.d/openflow stop

When connecting a new system to openflow we have to change the config file on the router:  vi /etc/config/openflow   and then change option 'ofctl' to 'tcp:INSERT_IPADDRESSOFSYSTEM'
Also make sure there's no firewall running:  sudo ufw status (to check)           then           sudo ufw disable   or sudo ufw allow 1191/tcp    (opwns 1191 port number to allow TCP traffic through ufw firewall)
